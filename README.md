# Library Management System
A command-line based Library Management System developed using Python.
This project demonstrates the practical implementation of Python concepts such as Object-Oriented Programmming, file handling, exception handling, and modular programming.
## Features
- Add book
- Remove book
- Search book by title
- Search book by author
- Issue book
- Return book
- Display available books
- Display issued books
## Concepts Used
### OOP
Uses classes and objects to represent books and manage library operations.
### File Handling
Book records are stored in files to maintain data even after the program is closed.
### Exception Handling
Handles invalid operations and user inputs using exceptions and custom exception classes.
### Modules and Packages
The project is divided into multiple Python files to improve code organization and reusability.
